全局 API

全局 API 是 Lynx 在 JavaScript 运行环境下提供用于使用框架基础能力的 JavaScript API，例如网络请求，开发者通过了解 API 的功能后自行使用。

全局 API 概述

名称 功能
Window JS 运行环境上下文，相当于 global，提供基础功能
Navigator 提供获取 UserAgent 功能
Screen 提供获取屏幕信息功能
Loader 提供网络请求功能
Console 提供各平台控制台接入
Document 提供构建和获取页面内容的能力
Element 元素，描述相同种类元素普遍具有的方法和属性。
